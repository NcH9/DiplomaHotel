## How to install this (don't)

- **Backend (!DOCKER REQUIRED!)** </br>
cd docker </br>
docker compose build </br>
docker compose up -d </br>
docker run --rm -v $(pwd):/app composer install </br>
cd .. </br>

- **Frontend** </br>
cd frontend </br>
npm install </br>
